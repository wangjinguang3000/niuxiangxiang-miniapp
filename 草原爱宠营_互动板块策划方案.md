## 一、项目背景与目标

当前小程序已具备：商品商城、线下赛事报名、金币经济体系、签到任务、合伙人分销等核心功能。
现需要新增 **线上互动板块**，让用户无需亲临草原也能参与人宠互动，提升日活和用户粘性。

### 两大核心板块

| 板块 | 一句话定义 | 核心玩法 |
|------|-----------|---------|
| 🐕 **遛狗狗广场** | 人宠短视频社区 | 上传视频 → 浏览/点赞/评论赚金币 |
| 🏆 **人宠互动预选赛** | 周期性线上评选赛 | 投稿参赛 → 投票打榜 → 排名奖励 |

## 二、遛狗狗广场（宠物短视频社区）

### 2.1 产品定位

一个以"晒狗"为核心的短视频Feed流，类似抖音极速版的宠物垂类。
核心差异：**看视频能赚金币，发视频也能赚金币**。

### 2.2 用户动线

进入广场 → 上下滑动看视频 → 点赞+1金币 | 评论+2金币 | 分享+5金币
         ↓
   点击"+"按钮 → 选择/拍摄视频 → 写标题+选话题 → 发布成功+20金币
         ↓
   别人看你的视频 → 你获得观看奖励(每10次观看+1金币)
         ↓
   视频被精选 → 额外+100金币

### 2.3 金币经济模型（遛狗狗专用）

| 行为 | 金币奖励 | 每日上限 | 说明 |
|------|---------|---------|------|
| 浏览视频 | +1/条 | 50条/天 | 需观看满5秒 |
| 点赞视频 | +1/次 | 30次/天 | 给自己点赞不计 |
| 发表评论 | +2/条 | 20条/天 | 需≥5个字 |
| 分享视频 | +5/次 | 5次/天 | 分享到微信群 |
| 发布视频 | +20/条 | 3条/天 | 审核通过后发放 |
| 视频被观看 | +1/10次 | 无上限 | 创作者被动收入 |
| 视频被点赞 | +2/次 | 无上限 | 创作者被动收入 |
| 每日登录广场 | +5/天 | 1次/天 | 首次进入广场 |

### 2.4 视频规格要求

- 时长：15秒 ~ 60秒
- 内容：必须包含人+宠物同框
- 格式：MP4，建议竖屏9:16
- 审核：先发后审（敏感词过滤 + 人工抽查）

## 三、人宠互动预选赛（线上评选赛）

### 3.1 产品定位

月度主题赛制，用户投稿人宠互动视频参加评选，由全站用户投票产生排名。
类似"萌宠选美"但强调 **人宠互动的趣味性和默契度**。

### 3.2 赛制设计

每月1日~20日：投稿期（上传视频报名参赛）
每月21日~28日：投票期（全站用户投票）
每月29日~30日：公示期（公布排名）
每月最后一天：发奖（金币+实物奖励）

### 3.3 月度主题示例

| 月份 | 主题 | 要求 |
|------|------|------|
| 7月 | 🏖️ 夏日戏水秀 | 狗狗玩水/游泳视频 |
| 8月 | 🎾 最佳接球手 | 狗狗接飞盘/球视频 |
| 9月 | 🍂 秋日漫步 | 人宠户外散步视频 |
| 10月 | 🎃 萌宠变装秀 | 宠物cosplay/装扮视频 |
| 11月 | 🤝 最佳默契 | 人宠配合互动挑战 |
| 12月 | 🎄 圣诞亲子装 | 人宠穿亲子装视频 |

## 四、技术架构

### 4.1 数据库集合设计（新增4个）

**videos（视频表）**
- _id, _openid, author{nickname,avatar}, title, videoUrl, coverUrl
- duration, tags[], views, likes, comments, shares, isFeatured
- contestId(关联赛事), status(active/reviewed/banned), createdAt

**comments（评论表）**
- _id, _openid, videoId, author{nickname,avatar}, content, likes, createdAt

**contests（赛事表）**
- _id, title, theme, rules, phase(submit|vote|done)
- submitStart/End, voteStart/End, prizes[], participantCount, totalVotes

**votes（投票记录表）**
- _id, _openid, contestId, videoId, date(YYYY-MM-DD), createdAt

### 4.2 云函数（新增5个）

| 云函数 | 功能 |
|--------|------|
| uploadVideo | 上传视频元数据到数据库 |
| getFeed | 获取视频Feed流（分页、排序） |
| interactVideo | 点赞/观看视频（含金币发放） |
| commentVideo | 发表/获取评论 |
| contestOps | 赛事操作（投稿/投票/查询） |

### 4.3 页面规划（新增3个）

| 页面路径 | 名称 | 功能 |
|---------|------|------|
| pages/feed/feed | 遛狗狗广场 | 视频Feed流 |
| pages/contest/contest | 预选赛 | 赛事列表+详情+投票 |
| pages/upload/upload | 上传视频 | 拍视频/选视频+发布 |

## 六、运营推广策略

### 6.1 冷启动
- 运营团队先上传20-30条高质量示范视频
- 首页Banner推广"发视频赚金币"
- 首期预选赛设高额奖励吸引参赛

### 6.2 日常运营
- 每日精选3条视频置顶推荐
- 每周"最佳铲屎官"评选
- 每月预选赛冠军采访+首页展示

### 6.3 裂变机制
- 分享视频到微信群→双方都得金币
- "帮我投票"分享卡片→拉新用户
- 参赛者自发拉票→免费传播
