## 一、小程序推广 vs 网站推广：本质区别

| | 网站 (niuxiangxiang.com) | 小程序 (草原爱宠营) |
|---|---|---|
| 战场 | 公域（百度/小红书/抖音引流到网站） | **微信私域**（在微信里完成一切） |
| 增长引擎 | SEO + 内容营销 → 搜索流量 | **分享裂变 + 微信群 + 公众号** |
| 用户获取 | 被动等待搜索 | 主动分享传播 |
| 变现方式 | 品牌展示 → 引导加微信 | 直接闭环：浏览→下单→支付→物流 |
| 核心竞争力 | 品牌故事 + 内容 | **金币经济 + 社交互动** |

> 💡 **结论**：网站和小程序是两个完全不同的战场，推广策略必须分开。
